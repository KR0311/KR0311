## 🏗️ KR0311 Control Panel (Core Project)

A fully custom hosting control panel designed to replace traditional solutions.

This is the primary long-term project, built from the ground up with a focus on automation, performance, and clean UX.

### 🔧 Current Features (In Progress)

* Hosting accounts & site provisioning
* Nginx + PHP-FPM automation
* SSL readiness & domain validation
* Client panel redesign (cPanel-inspired UX)
* Background job provisioning system (queue-based)
